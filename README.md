# stripo-plugins-charts

### How to add repo
```
helm repo add stripo 'https://raw.githubusercontent.com/stripoinc/stripo-plugins-charts/main/'
```

#### Update repo
```
helm repo update
```

#### Build charts
1. Make all changes in helm-chart-sources and run
2. Build charts:
```
bash release.sh
```
3. Commit changes and push to repo
